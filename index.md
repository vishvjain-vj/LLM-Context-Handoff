# Privacy Policy

Last Updated: June 2026

LLM Context Handoff helps users compress and copy AI chat conversations for migration between AI platforms.

## Information We Access

The extension accesses conversation content from the currently active AI chat page only when the user explicitly activates the extension.

## Gemini API Key

Users may optionally provide their own Google Gemini API key.

The API key is stored locally in the user's browser using Chrome Extension Storage and is used solely to make requests to Google's Gemini API on the user's behalf.

The API key is not transmitted to, stored by, or shared with the developer.

## Data Usage

Conversation content is processed only for generating a compressed context summary.

Conversation content and the user's Gemini API key are sent directly to Google's Gemini API when the user requests a context capsule.

## Data Storage

The extension stores:
- User-provided Gemini API key (locally in browser storage)

The extension does not store conversation content after processing.

## Third-Party Services

The extension communicates with Google Gemini API to generate context summaries.

Google's handling of submitted data is governed by Google's own policies.

## Third-Party Sharing

We do not sell, transfer, or share user data with third parties.
